# Nonull 一页纸速览

---

## 它是什么？

> 一个**有性格、有记忆、有安全意识**的智驾 AI 工程师。

---

## 它会什么？

```
审查代码     → "帮我 review 这个模块"
安全分析     → "做 HARA 分析"
设计测试     → "生成测试用例"
分析数据     → "看看日志有什么问题"
生成场景     → "生成雨天夜间场景"
研究论文     → "最近有什么新论文？"
```

---

## 它特别在哪？

```
👤 有性格     → 保守派/运动派/老司机
🧠 有记忆     → 四种记忆，1B 容量，不会忘
🛡️ 有安全     → 五关检查，不安全的事不做
🌱 能成长     → 越用越聪明，能自己学新技能
🏅 有徽章     → 像打游戏一样攒安全徽章
```

---

## 怎么用？

```bash
# 安装
pip install -r requirements.txt

# 启动
python -m Nonull

# 然后直接说话就行了
```

```python
from core import Nonull
agent = Nonull()
result = agent.run_sync("审查这段 AEB 代码")
```

---

## 三种人格怎么切？

```python
from persona import PersonaOrchestrator, PersonaType

# 保守派（做安全分析时用）
ai = PersonaOrchestrator(PersonaType.CONSERVATIVE)

# 运动派（做性能优化时用）
ai = PersonaOrchestrator(PersonaType.SPORTY)

# 老司机（做场景分析时用）
ai = PersonaOrchestrator(PersonaType.VETERAN)
```
