# Nonull 一页纸速览

---

> **📌 重要声明 / Important Disclaimer**
>
> Nonull 是一个**内部使用的 ADAS 工程开发助手**，**不是**经过 ISO 26262 / ASIL-D 认证的车规级安全产品。本项目中的"安全层"是**建议性（advisory）**的——它参考 ISO 26262 / MISRA / ASPICE 等标准的模式与术语进行风险提示和检查建议，但**并不实现** ASIL-D 要求的抗干扰、MC/DC 覆盖、形式化验证、SEooC 等条款。**请勿用于任何量产部署、安全关键决策，或替代经过认证的安全机制。**

---

## 它是什么？

> 一个**有性格、有记忆、有安全意识**的智驾 AI 工程师。

---

## 它会什么？

```
审查代码     → "帮我 review 这个模块"
安全分析     → "做 HARA 分析"
设计测试     → "生成测试用例"
分析数据     → "看看日志有什么问题"
生成场景     → "生成雨天夜间场景"
研究论文     → "最近有什么新论文？"
```

---

## 它特别在哪？

```
👤 有性格     → 保守派/运动派/老司机
🧠 有记忆     → 四种记忆，可配置容量（默认 10K 条目/层），不会忘
🛡️ 有安全     → 五关检查（advisory 建议性，非认证）
📊 有指标     → 安全操作记录与统计（建议性，非游戏化）
```

---

## 怎么用？

```bash
# 安装
pip install -e .
# 或
pip install -r requirements.txt

# 启动
python -m nonull
# 或
nonull

# 然后直接说话就行了
```

```python
from core import Nonull
agent = Nonull()
result = agent.run_sync("审查这段 AEB 代码")
```

## 三种人格怎么切？

```python
from persona import PersonaOrchestrator, PersonaType

# 保守派（做安全分析时用）
ai = PersonaOrchestrator(PersonaType.CONSERVATIVE)

# 运动派（做性能优化时用）
ai = PersonaOrchestrator(PersonaType.SPORTY)

# 老司机（做场景分析时用）
ai = PersonaOrchestrator(PersonaType.VETERAN)
```
